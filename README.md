# The Charlotte Grocery Gap: A Geospatial Accessibility Study

### The Problem
Does your ZIP code determine your access to fresh food? I compared "Premium" vs. "Budget" grocery distribution in Charlotte to identify potential food deserts.

### Key Discovery
* **Economic Disparity:** High-income areas like Ballantyne (28277) have a **6:1 ratio** of premium grocers to budget options.
* **The Gap:** West Charlotte (28208) showed **zero** premium options within the sample, highlighting a clear accessibility barrier for fresh, organic produce.

### Methodology
* **Data Source:** Manual Audit of Charlotte ZIP Code retail distribution.
* **Metrics:** Premium-to-Budget Ratio (PBR).
